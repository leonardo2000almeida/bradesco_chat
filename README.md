
# dashboard simples

Uma simples dashboard para demonstrar meus conhecimentos com react.

## Run Locally

Clone the project

```bash
  git clone  https://github.com/leonardo2000almeida/bradesco_chat.git
```

Go to the project directory

```bash
  cd bradesco_chat
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
